| DESCRIPTION   | SYNTAX                                        |
| -----------   | -----------                                   |
| Heading       | # H1 <br> ## H2 <br> ### H3                   |
| Bold          | \*\*TEXT\*\*                                  |
| Italic        |  \*TEXT\*                                     |
| Block Quote   | >TEXT                                         |
| Code block    | \```<br> enclose your code in this <br> ```   |



In Tables of Markdown format as created above , if you want column values to be aligned follow below format

 :----    = To align left  
 :----:   = To align centre  
 -----:   = To align right
